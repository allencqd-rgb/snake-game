# 🐍 贪食蛇

经典贪食蛇网页游戏，即开即玩，无需下载。

🔗 **在线玩**: https://allencqd-rgb.github.io/snake-game/

## 功能

- ⌨️ 键盘控制：方向键 / WASD
- 📱 移动端支持：虚拟方向键 + 滑动手势
- 🍎 吃食物得分，蛇身变长
- 💥 撞墙或撞自身游戏结束
- 🏆 最高分本地持久化（localStorage）
- ⏸️ 空格键暂停/继续
- ⚡ 每 50 分加速升级（共 5 级）

## 技术

纯前端实现，无外部依赖，单 HTML 文件。

- Canvas 2D 渲染
- 响应式布局
- localStorage 持久化

## 本地运行

```bash
git clone https://github.com/allencqd-rgb/snake-game.git
cd snake-game
# 直接打开 index.html 即可
```
